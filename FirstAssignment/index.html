Instruction to run the first assignmen as client
<=============================================================================>
In order to visualize a static page just open the browser and digit the follow url:

localhost:'port-number-choosen'/file-name

Make sure that the file required is present in the project folder
<=============================================================================>
In order to run a process and than visualize its result in a dynamic page, open the browser and digit the url below:

localhost:'port-number-choosen'/process/'process-name' 'param1'=value&'param2'=value

Make sure that the file required is present in the folder"process" located in the project folder
<=============================================================================>
